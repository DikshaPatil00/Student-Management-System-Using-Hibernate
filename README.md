# Student-Management-System-Using-Hibernate
An advanced web application built using the Hibernate ORM framework and Java Servlets. This project demonstrates a clean transition from manual JDBC to automated Object-Relational Mapping (ORM) for managing student data.

# Features
* Automated Persistence: Uses Hibernate to handle all database INSERT, UPDATE, and DELETE operations.
* Web Interface: Interactive JSP pages for adding, editing, and viewing student records.
* Session Management: Efficiently manages database sessions using a custom HibernateUtils utility class.
* Transaction Safety: Ensures data integrity by wrapping database operations in Hibernate Transactions.

# Tech Used
* Backend: Java (JDK 17)
* ORM Framework: Hibernate 
* Web Server: Apache Tomcat 
* Database: MySQL
